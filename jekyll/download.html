Nightly build notes

Simulationcraft nightly builds are created on a semi-regular schedule (roughly every day) for both Windows and MacOS. The nightly builds in the link above are always at the top of the listing (the list is sorted by descending date). The nightly builds contain a git commit hash of the source that was used to build the release. For example, simc-720-01-nightly-osx-x86-5d1a8e17c.dmg was built from the sources that contain all code up to this commit.
